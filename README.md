# AI Email Automation Agent (n8n + OpenAI)

## 🚀 Overview
This project is an AI-powered email automation workflow built using n8n.  
It uses OpenAI to generate and send professional emails automatically via Gmail.

## 🧠 Features
- AI email generation using GPT
- Automatic email sending via Gmail
- Chat-based trigger system
- Memory support for context awareness

## 🛠️ Tech Stack
- n8n (Workflow Automation)
- OpenAI API
- Gmail API

## 📂 How to Use
1. Import the JSON workflow into n8n
2. Connect your OpenAI credentials
3. Connect your Gmail account
4. Start the workflow and send messages

## ⚠️ Note
Credentials are not included for security reasons.  
You must add your own API keys.
